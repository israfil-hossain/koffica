## Asset Guidelines

### Image Specifications
- **Format**: PNG for graphics with transparency, JPG for photos
- **Quality**: High resolution, optimized for web
- **Style**: Modern, clean, coffee-themed aesthetic
- **Color Palette**: Warm browns, creams, blacks, with yellow accent (#e7d393)

### Video Specifications
- **Format**: MP4
- **Duration**: 10-30 seconds (loopable)
- **Resolution**: 1920x1080 minimum
- **Content**: Coffee brewing, pouring, or preparation process

### Leaf Elements
- Should be decorative botanical elements
- Transparent backgrounds (PNG)
- Organic, flowing shapes
- Complement the coffee theme

### Coffee Product Images
- Clean, professional product photography
- Consistent lighting and background
- Show coffee beans, cups, or brewing equipment
- High contrast for good visibility on dark backgrounds

## Implementation Notes

1. All images should be placed in the `/public/images/` directory
2. Videos should be placed in the `/public/videos/` directory  
3. Fonts should be placed in the `/public/fonts/` directory
4. Ensure all assets are optimized for web performance
5. Consider using Next.js Image optimization features
6. Test all assets on different screen sizes and devices

## Fallback Options

If specific assets are not available, you can:
1. Use placeholder images from services like Unsplash or Pexels
2. Create simple geometric shapes as temporary placeholders
3. Use CSS gradients or solid colors as backgrounds
4. Replace video with static images initially
